Módulos del Curso

1 Modelado de Datos Avanzado

Estructuración de esquemas en estrella y copo de nieve. Configuración de relaciones bidireccionales y filtrado cruzado. Creación de tablas de calendario y jerarquías personalizadas.

2 DAX para Análisis Real

Funciones de contexto: CALCULATE, ALL, FILTER. Medidas rápidas vs medidas personalizadas. Patrones comunes para KPI empresariales. Introducción a TIME INTELLIGENCE.

3 Visualizaciones Interactivas

Selección de gráficos por tipo de dato. Uso avanzado de tooltips y drillthrough. Bookmarks para navegación profesional. Formato condicional y paletas corporativas.

4 Publicación y Colaboración

Workspaces en Power BI Service. Configuración de actualizaciones programadas. Compartir dashboards con RLS (Row Level Security). Integración con SharePoint y Teams.

Actividad Práctica

Creación de Dashboard de Ventas

Implementa un dashboard interactivo con los siguientes componentes:

  1. Importa el dataset "Ventas_2024.xlsx" desde la carpeta de recursos
  2. Establece relaciones entre tablas de ventas, productos y clientes
  3. Crea medidas DAX para: YTD (Year-to-Date), crecimiento mensual y margen promedio
  4. Diseña 3 visualizaciones: gráfico de líneas (tendencia), matriz por categoría y mapa de calor regional
  5. Agrega dos bookmarks para vistas ejecutiva y detallada
  6. Aplica formato corporativo con colores #1a3a5c y #ffffff

Tiempo estimado: 25 minutos
