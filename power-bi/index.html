1

Interfaz de Power BI y Flujo de Trabajo

Power BI Desktop se organiza en tres vistas principales: Vista de Informes, Vista de Datos y Vista de Modelo. Cada vista cumple un propósito específico en el proceso de creación de business intelligence.

💡 Profesional Tip: Configura el tema oscuro en Archivo → Opciones → General para reducir fatiga visual durante sesiones prolongadas.

El panel de Visualizaciones (derecha) contiene todos los tipos de gráficos disponibles, mientras que el panel de Campos muestra las tablas y columnas de tu modelo de datos.

2

Importación y Transformación de Datos

Power BI se conecta a más de 100 fuentes de datos: Excel, SQL Server, SharePoint, APIs REST, etc. El proceso de transformación se realiza en Power Query Editor.

Pasos esenciales en Power Query:

  • Eliminar filas con valores nulos o errores
  • Cambiar tipos de datos (texto, número, fecha)
  • Crear columnas personalizadas con fórmulas M
  • Unir tablas (merge) y agregar datos (append)
⚠️ Importante: Siempre aplica transformaciones en Power Query, no en DAX. Esto optimiza el rendimiento del modelo.
3

Creación de Visualizaciones Profesionales

Selecciona visualizaciones según el tipo de análisis:

  • Tablas y Matrices: Para datos detallados
  • Gráficos de Barras/Líneas: Para tendencias y comparaciones
  • Mapas: Para análisis geográfico
  • Gauges y KPI: Para métricas de desempeño

Utiliza el formato para mejorar la legibilidad: ajusta colores corporativos, fuentes, títulos descriptivos y tooltips personalizados.

4

DAX Básico y Publicación de Reportes

DAX (Data Analysis Expressions) es el lenguaje de fórmulas de Power BI. Tres funciones esenciales para comenzar:

1. CALCULATE: Filtra cálculos en contexto específico
2. SUMX: Suma iterativa con lógica personalizada
3. RELATED: Trae datos de tablas relacionadas

Para publicar, haz clic en "Publicar" en la cinta de opciones. El reporte se subirá a tu workspace en Power BI Service, donde podrás configurar actualizaciones programadas, compartir con colegas y crear dashboards consolidados.

📝 Actividad Práctica: Crear un Reporte de Ventas

Sigue estos pasos para aplicar lo aprendido:

  1. Importa un archivo Excel con datos de ventas (columnas: Fecha, Producto, Región, Cantidad, Valor)
  2. En Power Query, crea una columna "Trimestre" basada en la fecha
  3. Crea un gráfico de líneas que muestre la evolución de ventas por trimestre
  4. Añade una tabla con ventas totales por región
  5. Crea una medida DAX: "Ventas Promedio = AVERAGE(Ventas[Valor])"
  6. Agrega un card visual mostrando esta medida
  7. Aplica formato corporativo (colores, logos, fuentes)
